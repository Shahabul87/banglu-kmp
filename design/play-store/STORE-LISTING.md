## Full Description (4000 chars max)

🇧🇩 **Banglu — The Smartest Bengali Keyboard**

Type Bengali effortlessly using English letters! No need to learn a new keyboard layout — just type how you speak, and Banglu converts it to perfect Bengali instantly.

**✨ How It Works**
Type "ami banglay gan gai" → আমি বাংলায় গান গাই
That's it! Banglu's 7-layer AI engine understands your phonetic input and produces the correct Bengali text.

**🧠 Smart Engine**
• 485,000-word Bengali dictionary — the largest in any keyboard
• AI disambiguation: automatically picks the right ত/ট, ন/ণ, র/ড়, শ/ষ
• Real-time suggestions with swap variants
• Next-word predictions based on context
• Learns your typing habits and preferences

**⌨️ Samsung-Quality Keyboard**
• Beautiful dark theme with smooth animations
• Number row always visible
• Emoji panel with 500+ emoji
• Double-space → period
• Auto-capitalize after sentences
• Swipe spacebar for cursor movement
• Long-press numbers for quick symbols (1→!, 2→@, 3→#)
• Hold backspace for fast word-by-word deletion
• Context-aware Enter key (🔍 for search, ➤ for send)

**🌐 Dual Language — One Keyboard**
Switch between Bengali and English with one tap. No need to change keyboards!
• Bengali mode: Smart phonetic conversion
• English mode: Direct typing like any standard keyboard
• Seamless switching with the EN/BN button

**🔒 Privacy-First**
• Core phonetic typing works on your device
• NO keystroke logging for ads or analytics
• NO ads — ever
• Optional voice typing uses your Android speech provider
• Optional account sync keeps your custom dictionary available across devices

**📱 Works Everywhere**
Type Bengali in WhatsApp, Facebook, Messenger, Chrome, Gmail, Notes — any app!

**🎯 Common Conversions**
ami → আমি | tumi → তুমি | bhalo → ভালো
bangladesh → বাংলাদেশ | sundor → সুন্দর
khabar → খাবার | dhonnobad → ধন্যবাদ
