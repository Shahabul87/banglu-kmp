## Description
যেকোনো লেখার ঘরে ইংরেজি অক্ষরে লিখুন, Space চাপুন — বাংলা হয়ে যাবে।
Gmail, Facebook, WhatsApp Web, GitHub — সব জায়গায়।

• ami banglay likhi → আমি বাংলায় লিখি
• চ্যাটের বানানও বোঝে: issa → ইচ্ছা, kmon → কেমন, korsi → করছি
• শব্দ লেখার সময় সাজেশন চিপ — ক্লিক করলেই সেই বানান
• Alt+B — যেকোনো সাইটে এক চাপে চালু/বন্ধ
• পপআপ কনভার্টার — যেখানে ইনলাইন কাজ করে না (যেমন Google Docs), লিখে কপি করুন

প্রাইভেসি: এক্সটেনশনটির কোনো নেটওয়ার্ক অনুমতি নেই — অভিধানসহ সবকিছু
আপনার ব্রাউজারের ভেতরে চলে। আপনার লেখা কোনো সার্ভারে যায় না, কখনোই।

এটি Banglu Android কীবোর্ডের সেই একই ইঞ্জিন — একই মানের বাংলা, এখন ডেস্কটপে।

## Privacy practices (CWS questionnaire)
- Single purpose: convert romanized Bengali typing to Bangla script locally
- Permissions: storage (per-site on/off preference only)
- Host access <all_urls>: required to convert typing in text fields on any
  site the user chooses; no data is read beyond the word being typed, and
  nothing is transmitted (the extension has no network capability)
- No remote code; no analytics; no data collection of any kind
